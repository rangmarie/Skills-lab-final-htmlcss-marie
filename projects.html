Best UXD projects

Adidas

This project was for a class called Research for Design where we had Adidas as a client. It was a very open assignment targeted to the official mobile website of adidas with a target group of 16-22 year olds. We were given three keywords to focus on: premium, connected and personalised.

My main HMW questions were:

  • How might we make users associate the “premiumness” associated with the whole brand associated only with the official website and app?
  • How might we provide the users with a sense of immediate and delayed satisfaction that is incomparable to money?

Programming

In our course for basic Javascript, HTML and CSS, we had 3 main assignments, which were all related to space travel, specifically to Mars. The first assignment was to create an interface that shows information about the flight to Mars. The second assignment was to create a clock interface with some sort of animation and the third was to create a google maps API for where the ship that went to Mars could possibly land on its way back.
